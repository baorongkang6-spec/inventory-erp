## 一、日常使用

### 1.1 界面导航

登录后顶部菜单：

| 菜单 | 内容 |
|---|---|
| **总览表** | 三家公司 银行/库存/往来/票据 的期初·收入·发出·结存汇总 |
| **基础资料** | 商品、客户、供应商、费用类别、银行账户 |
| **采购** | 采购入库、采购发票、付款 |
| **销售** | 销售出库、销售发票、收款 |
| **库存报表** | 各商品结存数量与金额 |
| **查询** | 跨公司组合查询中心 |
| **资金** | 其他收支登记、应收票据、应付票据 |
| **报表** | 各类余额表、日记账、成本表、对账、期初导入 |
| 右上「**当前账套**」 | 切换当前操作的公司（C1/C2/C3） |
| 右上「**用户名**」 | 修改密码、用户管理(管理员)、后台管理 |

> ⚠️ **操作前先看右上"当前账套"是哪家公司**。录单、看报表都针对当前账套。切错账套会记到错公司。

### 1.2 角色与权限
系统按岗位分角色，看到的菜单/能做的操作不同：
- **总经理**：看全部报表、跨公司总览。
- **出纳**：银行、收付款、票据。
- **采购**：采购入库、采购发票。
- **销售**：销售出库、销售发票。
- **财务**：发票、核销、报表、期初、对账。

---

### 1.3 基础资料（先建好，才能录单）

**基础资料** 菜单下：
- **商品**：编码、名称、单位。录入库/出库前必须先建商品。
- **客户 / 供应商**：往来单位主数据。
- **银行账户**：每家公司的银行账户（含期初余额）。
- **费用类别**：采购/销售单上"其他费用"用到的类别（如运费）。

> 每家公司的基础资料各自独立，按"当前账套"维护。

---

### 1.4 采购流程（进货）

**第 1 步 采购入库**（采购 ▸ 采购入库 ▸ 新建入库单）
- 选供应商、日期；
- 商品明细：选商品、填**数量**和**含税单价**，系统**自动算**出含税金额、不含税金额、税额（可手工微调税率）；
- 有运费等可在"其他费用"填（可选择是否计入成本）；
- 点 **「保存并过账」**（⚠️ 回车不会提交，必须点这个按钮）。
- 过账后**库存立即增加**，成本按**移动加权平均**计算。

**第 2 步 收到发票**（入库单列表/详情 ▸「收到发票」）
- 点「收到发票」会跳到采购发票录入页，并**自动带出入库单明细**；
- 核对后登记 → 自动形成对该供应商的**应付账款**；
- 当某入库单的货**全部收到发票**后，列表上会显示「**发票已收**」。
- 还没收到票的，可用报表 **「已入库未收票明细表」** 查，作为**库存暂估**依据。

**第 3 步 付款**（采购 ▸ 付款）
- 登记付款 → **自动生成银行存款日记账**（银行存款减少）；
- 在付款详情里可把这笔款**核销**到具体的采购发票（支持部分核销）。

---

### 1.5 销售流程（出货）

**第 1 步 销售出库**（销售 ▸ 销售出库 ▸ 新建出库单）
- 选客户、日期；商品明细填数量、含税单价（自动算金额/税额）；
- 点 **「保存并过账」** → 库存立即减少，并按移动加权**结转销售成本**。

**第 2 步 开具发票**（出库单列表/详情 ▸「开具发票」）
- 自动带出出库明细（含数量与成本关联）→ 登记 → 形成对该客户的**应收账款**；
- 全部开票后显示「**发票已开具**」；
- 未开票的可用 **「已出库未开票明细表」** 查。

**第 3 步 收款**（销售 ▸ 收款）
- 登记收款 → 自动生成银行日记账（银行存款增加）；
- 在收款详情里**核销**到具体销售发票。

---

### 1.6 关联交易自动联动（三家公司之间）
当一家公司**销售出库**给关联公司时，系统会在对方公司**自动生成对应的采购入库单**（镜像），无需双方各录一遍。源单作废时，镜像单也会联动作废并反冲库存。

---

### 1.7 资金与票据
- **其他收支登记**（资金 ▸ 其他收支登记）：与发票无关的零星银行收支（手续费、利息等）。
- **应收票据 / 应付票据**（资金菜单）：登记收到/开出的票据；可冲应收、背书抵应付。
- **银行对账**（报表 ▸ 银行存款日记账 ▸ 银行对账）：导入银行流水核对。

---

### 1.8 报表（都支持多公司联合查询 + 导出 Excel）

报表菜单各表用途：
| 报表 | 看什么 |
|---|---|
| 账户余额表 | 银行/库存/应收/应付 各账户期初·本期增减·期末 |
| 银行存款日记账 | 每个银行账户逐笔收支与余额（账户可选"全部账户"） |
| 应付账款余额表 | 各供应商欠款：期初/本期增加/本期减少/期末；**点供应商名进明细账** |
| 应收账款余额表 | 各客户应收：同上；**点客户名进明细账** |
| 销售收入成本表（按开票/按出库） | 销售收入与对应成本、毛利 |
| 已出库未开票明细表 | 出库了但还没开发票的（核对开票） |
| 已入库未收票明细表 | 入库了但还没收到票的（库存暂估依据） |
| 票据余额表 | 在手未用的应收/应付票据 |
| 借调往来余额表 | 关联公司间借调往来 |
| 月底对账 / 对账历史 | 月末勾稽与历史留存 |

- 每张报表上方可勾选**公司**（不选=全部）、选**日期区间**，点「查询」。
- 右上「**导出 Excel**」可下载（带表头、编制单位、期间、合计）。

---

### 1.9 期初导入（首次启用时）
**报表 ▸ 期初导入**：
1. 下载模板（一个 Excel 工作簿，含银行/库存/应收/应付等多个 Sheet）；
2. 按 Sheet 填好各公司期初数据（库存填数量+金额）；
3. 上传导入。
> 期初基准日：2026-06-01。期初导入一般由财务在启用时做一次。

---

### 1.10 修改 / 作废单据（重要规则）
- **采购入库 / 销售出库**：详情页有「**修改**」「作废」。可改条件：**本人或管理员** + **本月内** + **未被下游引用**（已开票/已镜像的不能直接改）。
- **发票**：可「作废」；销售发票可「修改」重新关联出库单。
- ⚠️ **移动加权特别提醒**：如果一批入库的货**已经被后续出库卖掉一部分**，这张入库单就**不能直接改/作废**了（系统会提示"已被后续出库消耗"）。需要先**作废相关的出库单**，再回来改入库单。这是移动加权成本核算的正常限制，不是故障。

---

### 1.11 修改自己的密码
右上「用户名 ▸ 修改密码」，输旧密码 + 新密码两遍即可。

## 二、管理员运维（系统管理员看）

> 服务器：公司台式机 `WIN-2EVCS3G915C`，内网 IP `192.168.10.245`。
> 代码目录：`C:\inventory-erp`。命令都在**管理员 PowerShell**里、先 `cd C:\inventory-erp`。

### 2.1 服务管理（开机自启服务名 InventoryERP）
```powershell
.\nssm.exe status InventoryERP     # 查看状态（SERVICE_RUNNING 为正常）
.\nssm.exe restart InventoryERP    # 重启
.\nssm.exe stop InventoryERP       # 停止
.\nssm.exe start InventoryERP      # 启动
Get-Content C:\inventory-erp\logs\service.log -Tail 50   # 看运行日志（排错用）
```
> 服务已设开机自启，台式机重启后 ERP 会自动运行，无需人工干预。**这台机平时不要关机**。

### 2.2 新增用户 / 分配角色（仅管理员）
右上「用户名 ▸ 用户管理」：
- 新增人员、设置登录账号；
- 给每人勾选所属公司和**角色**（财务/采购/销售/出纳/总经理）；
- ⚠️ **每个同事用各自账号**，不要共用 admin —— 这样权限可控、操作有审计记录。

### 2.3 每日备份（财务数据命脉，务必做）
SQLite 数据库就是一个文件，备份=拷贝它：
```powershell
Copy-Item C:\inventory-erp\db.sqlite3 D:\erp备份\db_$(Get-Date -Format yyyyMMdd).sqlite3
```
- 路径 `D:\erp备份` 换成你的备份盘/U盘；
- 建议**每天一次**，并**定期再拷一份到 U盘/网盘**（异地保存）；
- 重要：偶尔做一次"恢复演练"，确认备份能用。

### 2.4 远程更新（开发者改了功能后）
开发者在 Mac 端推送后，台式机执行：
```powershell
cd C:\inventory-erp
git pull
uv sync
uv run python manage.py migrate
uv run python manage.py collectstatic --noinput
.\nssm.exe restart InventoryERP
```

### 2.5 访问与网络

- 如需更高安全，可在花生壳「访问控制」加 IP 白名单或访问口令。

## 三、常见问题（FAQ）

**Q：录单时点回车没反应？**
A：正常。录入单据**必须点「保存并过账」按钮**，回车不会提交（防误操作）。

**Q：报表里数字不对/看不到某公司？**
A：先看右上"当前账套"，并在报表上方勾选要看的公司、选对日期区间，再点查询。

**Q：入库单改不了，提示"已被后续出库消耗"？**
A：这批货已被出库卖掉一部分，需先作废相关出库单再改。见 1.10。

**Q：台式机自己的浏览器打不开 127.0.0.1:8000？**
A：不影响使用。服务器自己不用开页面，大家用内网 IP 或外网域名即可。

**Q：外网打不开了？**
A：①确认台式机开着、服务 RUNNING（2.1）；②确认花生壳客户端在线；③内网 IP 若变化需在花生壳更新。
