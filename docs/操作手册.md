## 一、日常使用

### 1.1 界面导航
顶部菜单：**总览表 / 基础资料 / 采购 / 销售 / 费用 / 库存报表 / 查询 / 资金 / 报表**；右上「当前账套」切换公司（C1/C2/C3）；右上用户名 → 修改密码 / 用户管理 / 后台管理。

> ⚠️ **录单、看报表前先看右上"当前账套"是哪家公司**。切错账套会记到错公司。

### 1.2 角色与权限
按岗位分角色（总经理/出纳/采购/销售/财务），看到的菜单与能做的操作不同。**佣金及含佣金的报表只有总经理(GM)能看**。

### 1.3 基础资料（先建好，才能录单）
**基础资料**菜单：商品、客户、供应商、费用类别、银行账户。每家公司各自维护。

**关联企业要点**：客户/供应商若是系统内的关联公司（C1/C2/C3 互相），要在其「对应关联企业」里选上——这样关联交易自动联动、利润表内部抵销才认得出来。

**一键复制基础资料到其他公司（管理员）**：基础资料 ▸ 复制到其他公司。在一家维护好后，把商品/客户/供应商/费用类别一键复制到另两家（编码名称一致；已存在的自动跳过、可反复点）。建议先在 C1 建全 → 复制到 C2/C3。

### 1.4 采购流程（进货）
1. **采购入库**（采购 ▸ 采购入库 ▸ 新建）：选供应商、日期；填数量、含税单价（自动算金额/税额）；点 **「保存并过账」**（回车不提交）→ 库存增加、移动加权计成本。
2. **收到发票**（入库单 ▸「收到发票」）：自动带出明细 → 登记 → 形成应付；全部收票后显示「发票已收」。发票上可填**账期(天)**。
3. **付款**（采购 ▸ 付款）：选**付款方式**——
   - **银行账户**：登记付款 → 自动生成银行日记账；详情里核销到具体发票（可部分、可超额）。
   - **应收票据（背书）**：用手上一张在手应收票据背书抵付应付（不生成银行日记账）。输**票据号码**自动带出出票日/到期日/可用余额；填**付款金额**（≤余额，可部分背书）；下方勾选要抵付的采购发票（合计=付款金额）。提交后票据置「已背书」。

### 1.5 销售流程（出货）
1. **销售出库**（销售 ▸ 销售出库 ▸ 新建）：选客户、日期、数量、含税单价 →「保存并过账」→ 库存减少、按移动加权结转成本。
2. **开具发票**（出库单 ▸「开具发票」）：带出明细 → 形成应收；全部开票后显示「发票已开具」。可填**账期(天)**。
> **收款/付款列表是统一一览**：列首「收款方式 / 付款方式」既显示银行账户，也显示**应收票据**——收到的客户票据、用票据背书付款都会并列在这里（带蓝色「应收票据」标记）。票据行可就地「修改」补录信息；冲销/作废请到「资金 ▸ 应收票据」。

3. **收款**（销售 ▸ 收款）：选**收款方式**——
   - **银行账户**：自动生成银行日记账（收入）；详情里核销到发票。
   - **应收票据**：收到客户票据 → 生成一张在手应收票据（不生成银行日记账）。**付款客户**必填，填**票据号码**、出票日期、到期日期、票面金额；下方可直接勾选要冲抵的销售发票（冲应收账款）。
   > ⚠️ **重要口径**：收到客户票据抵应收账款 = 借应收票据 / 贷应收账款——**冲应收账款只减少应收账款，不消耗票据**。这张票仍是你「**持有的应收票据（在手、未用=票面）**」，之后可整张**背书给供应商**或**到期托收进银行**。即「冲应收」是票"进来"，「背书/托收」才是票"出去"消耗未用额。
   > 票据后续到期兑付/贴现、背书抵应付，到「资金 ▸ 应收票据」操作。
   - **到期兑付**：票据到期，承兑行付款进银行。点应收票据列表的「到期兑付」→ 选收款银行账户、日期、票面金额（默认未用额）→ 票面进银行存款（自动生成银行日记账，业务类型「票据兑现」），票据消耗。分录：借 银行存款 / 贷 应收票据。
   - **贴现**：到期前卖给银行提前拿钱。点「贴现」→ 选银行账户、日期、票面金额、**实收净额**（银行实际到账）→ 实收净额进银行存款，**贴现息=票面−实收**自动记一笔财务费用，票据消耗。分录：借 银行存款(净额) + 财务费用(息) / 贷 应收票据(票面)。
   - 兑付/贴现都可在「票据使用明细」里点「撤销」回退（恢复票据未用额 + 删对应银行日记账/财务费用）。

### 1.6 账期与逾期
- 采购/销售发票录入有「**账期(天)**」：0=即期；**到期日 = 开票日 + 账期**。
- 发票详情显示到期日，过期未结显示**「已逾期」**。
- 逾期可在报表「**逾期明细表**」「应收/应付账款余额表（含账龄）」查看。

### 1.7 费用（佣金/销售/管理/财务费用）
顶部 **费用** 菜单：
- **佣金**（仅总经理）：录入日期、客户、产品、人员、金额、备注；总经理专属，其他角色看不到。
- **销售费用 / 管理费用 / 财务费用**：同样的录入界面（总经理/财务/出纳可用）。

### 1.8 关联交易自动联动（三家公司之间）
一家公司销售出库给关联公司（客户的"对应关联企业"已设）时，系统会在对方公司**自动生成采购入库单**（镜像），并自动带上代表本公司的供应商。源单作废会联动作废。商品按编码在对方自动配齐。

### 1.9 资金与票据
资金菜单：其他收支登记、应收票据、应付票据。银行对账在「报表 ▸ 银行存款日记账 ▸ 银行对账」。
- **应收票据可「修改」补录**：「资金 ▸ 应收票据」列表（及收款/付款统一一览中的票据行）有「修改」按钮，可补录票据号、出票日、到期日、来源客户、备注等（导入或漏填的票据尤其常用）。**已使用**（已部分冲应收/背书）的票据，描述字段仍可改，但**票面金额锁定**——改额会破坏已用/未用勾稽，需作废后重录；**已作废**的票据不可改。
- **撤销票据冲销**（更正误用的票据）：发票详情的「核销明细」中，「应收票据冲销 / 背书抵付 / 应付票据抵付」行可点「撤销」；票据使用明细里每笔使用也可「撤销」。撤销后**退回本发票未核销额、恢复票据未用额、票据回到「在手」**，并删除该冲销记录（记审计）。用于"误把收到的客户票据当作核销/背书用掉"等更正。
- **列表底部合计行**：应收票据（期初金额/本期收入/已用/未用）、收款登记、付款登记（金额/已核销/未核销）、**销售出库（不含税售额/含税售额/结转成本）** 列表底部均有一行**合计**，随当前筛选/日期区间变化。
- **销售出库列表加「不含税售额」列**：在含税售额左侧显示不含税销售额（导出 Excel 同列）。合计只汇总金额列，总数量异构不跨单相加。
- **采购入库列表加「不含税合计」列 + 合计行**：入库成本/不含税合计/含税合计列底部合计（对称销售出库）。
- **应收票据「票面」拆为「期初金额 / 本期收入」**：期初导入的票据（is_opening）金额计入**期初金额**列，本期出票的计入**本期收入**列（导出 Excel 同样两列）。
- **应收票据「已用」可点查使用明细**：列表里票据的「已用」金额若 > 0，点它直接进**该票据使用明细**（显示出票 + 每笔冲应收/背书抵应付的日期、被冲发票号，可点进发票），**看全部、不受日期区间限制**。用于回答"这张票的已用额是什么时候、冲了哪张发票"。
- **应收票据可「删除」**（录错可彻底移除）：「修改」旁有「删除」按钮，**只要未使用**（未冲应收、未背书抵应付）即可删——**含期初导入的票据**（期初票据恰恰最易录错），删除是干净的（该票据不挂应收/应付、无银行日记账、无镜像）。**已使用**的票据不出现删除按钮，如需更正请先到对应发票撤销冲销再删。收款统一一览中的票据行同样可删。
- **其他收支**（费用/税费/工资/内部划转等非往来收支）登记后在「银行存款日记账」该行操作列可「修改」「删除」（仅手工登记、**当月**、未对账的）。
- **收款 / 付款详情页**顶部也有「修改」「删除」（与列表同条件：当月、未核销、未对账）。

### 1.10 修改 / 作废单据（规则）
- 采购入库 / 销售出库有「修改」「作废」：**本人或管理员 + 本月内 + 未被下游引用**；列表页操作列也有「修改」快捷入口。
- **采购入库**还有「删除」（彻底移除并反冲库存）：因移动加权成本是链式的，**仅当该入库后该商品再无其它出入库变动**（即"刚录错、马上撤"）、且非镜像 / 未开票 / 当月 / 本人或管理员时才出现；其余情况请用「作废」（反冲库存、留痕）。
- **采购发票 / 销售发票**都可「修改」「作废」「删除」：**未核销 + 非期初**方可操作；列表页操作列有「修改」「删除」。
  - **作废**=保留记录、标「已作废」、从应收/应付剔除（留痕，推荐用于真实但需取消的发票）。
  - **删除**=彻底移除该发票及明细、撤销其应收/应付、不可恢复（用于录错的发票）；已作废且未核销的也可再删除清理。
- **收款 / 付款**（银行方式）在列表操作列有「修改」「删除」：**仅当月 + 未核销 + 未银行对账**才显示。修改会同步更新对应银行日记账；删除会一并删掉该日记账（不可恢复）。已核销的请先到详情撤销核销。
- **付款**修改时还可把「付款方式」改成「**应收票据（背书）**」：保存后会**删除原银行付款及其日记账**，改记为用该票据背书抵应付（适合「当初误记成银行付款、实为票据背书」的更正）。
- **收款**修改时同样可把「收款方式」改成「**应收票据**」：保存后会**删除原银行收款及其日记账**，改记为收到一张应收票据（可顺带勾选销售发票冲应收）。
- **可填负数（红字）发票**：金额可正可负（退货红冲），不能为 0；核销也可超过发票额（预收/预付）→ 应收/应付可为负。
- **库存可为负**：销售出库可超过结存（先出后入），结存可负。

## 二、报表（多数支持多公司联合查询 + 导出 Excel）

报表菜单各表：
| 报表 | 看什么 |
|---|---|
| 账户余额表 | 银行/库存/应收/应付各账户 期初·本期增减·期末；每个分组底部有**合计**行 |
| 银行存款日记账 | 各银行账户逐笔收支与余额（账户可选"全部账户"） |
| 应付账款余额表 | 各供应商：期初/增/减/期末 + **逾期金额 + 账龄**；点名进明细账 |
| 应收账款余额表 | 各客户：同上 + 逾期/账龄；点名进明细账 |
| 逾期明细表 | 逐张逾期发票（应收/应付）+ 逾期天数 + 账龄段 |
| 销售收入成本表（按开票/按出库） | 销售收入、成本、毛利（按出库表含**佣金列**，仅总经理） |
| **客户销售分析表（出库）** | 按客户[/产品] 销售数量/收入/成本/佣金/毛利率；可勾"按产品明细"，有客户小计、公司合计 |
| **管理利润表（出库）**（仅总经理） | 每公司本期/本年 收入-成本-佣金-利润；可勾**内部交易抵销** |
| **剩余发票额度查询** | 各公司 本年开票/本月开票/额度/剩余可开（不含税） |
| 费用汇总表 | 按 类别 + 维度(月/人员/客户/产品) 汇总（佣金类仅总经理） |
| 已出库未开票 / 已入库未收票明细表 | 开票/收票核对、库存暂估依据 |
| 票据余额表 / 借调往来余额表 | 在手票据 / 关联借调往来 |
| 月底对账 / 对账历史 | 月末勾稽与留存 |
| 期初导入 | 见 1.12 |

通用：报表上方勾选**公司**（不选=全部）、选**日期区间**；金额两位小数+千分位；右上「导出 Excel」带集团/公司抬头与 logo。

> **开票额度**：先在「基础资料 ▸ 开具发票额度录入」给各公司录每月额度，再用「剩余发票额度查询」看本月还能开多少。

### 1.12 期初导入与启用锁定（首次启用时）
报表 ▸ 期初导入：
1. 「下载模板」（一个 Excel 多 Sheet）→ 各 Sheet 填好（库存为数量+金额；各类金额**可填负数**）→ 上传导入。
2. 录错了？**未启用**时点「**清空期初**」（仅设置阶段、还没日常业务时）→ 改好 Excel → 重新导入。
3. 核对无误 → 点「**启用期初（锁定）**」。锁定后期初不可改/重导；需修改由**管理员**「解锁期初」。
> 启用日 2026-06-01。期初数据在总览/余额表里始终算"期初"列。

## 四、常见问题（FAQ）
- **录单点回车没反应？** 正常，必须点「保存并过账」。
- **报表看不到某公司/数字不对？** 看右上"当前账套"，并在报表上方勾公司、选日期区间。
- **入库单改不了，提示"已被后续出库消耗"？** 现已允许负库存，一般可改；如仍受限按提示作废下游单。
- **总览表期初是 0、数据跑到本期了？** 已修复——期初按"期初标记"恒计期初；若仍异常截图给管理员。
- **更新后没生效？** 确认 `update.bat` 的 `git pull` 成功（SSH 免密），再 Ctrl+F5。
